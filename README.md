### bellafetch<pre><code>my personal system information gathering tool<br></code>


![](https://github.com/xorsirenz/bellafetch/blob/main/ss.png?raw=true)

#### Prerequisites:
- golang

####  Installation:
##### Linux
```sh
# clone the repo and build the application
$ git clone https://github.com/xorsirenz/bellafetch.git 
$ cd bellafetch/
$ chmod +x build.sh
# it's required to run the build script as root to generate a binary in /usr/local/bin
$ sudo ./build.sh
# run
$ bellafetch
```
